香蕉天堂

主程式 3.81C

  • 下載連結：主程式下載
  • 請下載 3.81C 版本的主程式
  • 建議安裝目錄：C:\Program Files，例如 C:\Program Files\天堂（Lineage 3.81C）

登入器

  • 下載連結：登入器下載
  • 請先下載 Bananania登入器
  • 解壓縮後，直接覆蓋到主程式目錄 C:\Program Files\天堂（Lineage 3.81C）

開始遊戲

  • 執行主程式 C:\Program Files\天堂（Lineage 3.81C）\Login.exe

遊戲倍率

  • 經驗 x30
  • 金幣 x 5
  • 衝武器，防具 20%

遊戲說明

  • 角色創建獎勵，香蕉系列武器
  • 螢幕解析度，建議設置為 1024x768 或 1280x960
  • 解決打字閃的問題，請安裝自然輸入法 V10 版本，參考 安裝說明，自然輸入法 V10 官方下載點
  • 瞬移太快會卡點問題，請玩家先不要連續按瞬移或回捲，卡點造成無法操作時，需使用集體傳送或由 GM 招喚，或按下 Ctrl+Alt+Delete 強制關閉 Lineage.exe，再重新開啟即可 (TBC)

P2E 邊玩邊賺

  • 偷偷告訴你，遊戲中 BOSS 都藏有香蕉幣
  • 不定期舉辦活動，獵殺 BOSS
  • 衝等獎勵，送你香蕉幣
  • 列榜排名，獎勵香蕉幣

贊助

  • Bananania NFT

  • BANANA Coin

贊助說明

  • Bananania NFT 獎勵，耀武系列武器，+9 耀武系列武器還可再進行煉化，最終能進化成，超特製的耀武系列武器
  • Bananania NFT 是個人收藏，在延伸伺服器或改版，仍可繼續使用，並保留強化數值

  • 可跟 "香蕉GM" 以 1000 個 BANANA Coin 兌換 1 個香蕉幣，使用香蕉幣在遊戲中的商人那裡可以兌換高級防具、特殊藥水、物品
  • 身上的香蕉幣也可跟 "香蕉GM" 換回 BANANA Coin ，1 個香蕉幣可兌換 1000 個 BANANA Coin
  • BANANA Coin 是個人財產，在延伸伺服器或改版，仍可繼續使用

  • 使用 MetaMask 錢包，在 uniswap v3 官網上，使用 ETH 或 WETH，兌換成 BANANA Coin
  • 使用 MetaMask 錢包，在 uniswap v3 官網上，可將 BANANA Coin 兌換回 ETH 或 WETH

加密貨幣與區塊鏈 NFT 說明

  • 兌換 BANANA Coin 前，需先註冊加密貨幣交易所，如幣安，MAX，MaiCoin，使用法幣 (就是新台幣) 購買 ETH 以太幣
  • 下載 MetaMask，創建錢包，將 ETH 從加密貨幣交易所傳送到你的 MetaMask 錢包，需使用 ERC20 通道傳送
  • 透過 MetaMask 錢包，在 uniswap v3 官網上，使用 ETH 或 WETH ，兌換成 BANANA Coin

  • 購買 Bananania NFT，需透過 MetaMask 錢包，點擊連結，需使用 ETH 購買

  • 需特別注意，請選用有品牌的加密貨幣交易所，如幣安，MAX，MaiCoin
  • 需特別注意，網址是否正確，如小狐狸錢包 MetaMask : "https://metamask.io/"，加密貨幣交易所 MaiCoin : "https://www.maicoin.com/"
