香蕉天堂

主程式 3.8.1C

  • 下載連結：主程式下載
  • 請下載 3.81C 版本的主程式
  • 建議安裝目錄：C:\Program Files，例如 C:\Program Files\天堂（Lineage 3.8.1C）

登入器

  • 下載連結：登入器下載
  • 請先下載 Bananania登入器
  • 解壓縮後，直接覆蓋到主程式目錄 C:\Program Files\天堂（Lineage 3.8.1C）

開始遊戲

  • 執行主程式 C:\Program Files\天堂（Lineage 3.8.1C）\Login.exe

倍率

  • 經驗 x20
  • 金幣 x20

角色創建獎勵

  • 耀武系列武器

P2E 邊玩邊賺

  • 偷偷告訴你，遊戲中很多 BOSS 都藏有香蕉幣
  • 不定期舉辦活動，獵殺 BOSS
  • 衝等獎勵，送你香蕉幣

贊助

  • Bananania NFT

  • Bananania Coin

贊助說明

  • Bananania NFT 獎勵，超特製的耀武系列武器
  • Bananania NFT 是個人收藏，在延伸伺服器或改版，仍可繼續使用，並保留強化數值

  • Bananania Coin 兌換香蕉幣，1000 個 Bananania Coin 可兌換 1 個香蕉幣，使用香蕉幣在遊戲中的商人那裡可以兌換高級防具、特殊藥水、物品等
  • 身上的香蕉幣也可跟 "香蕉GM" 兌換成 Bananania Coin ，1 個香蕉幣可兌換 1000 個 Bananania Coin
  • Bananania Coin 是個人財產，在延伸伺服器或改版，仍可繼續使用

  • 使用 MetaMask 錢包，在 uniswap v3 官網上，使用 ETH 或 WETH，兌換成 Bananania Coin
  • 使用 MetaMask 錢包，在 uniswap v3 官網上，將 Bananania Coin 兌換成 ETH 或 WETH
  • Bananania Coin 是個人財產，在延伸伺服器或改版，仍可繼續使用

加密貨幣與區塊鏈 NFT 說明

  • 兌換 Bananania Coin 前，需先註冊加密貨幣交易所，如幣安，MAX，MaiCoin，使用法幣 (就是新台幣) 購買 ETH 以太幣
  • 下載 MetaMask，創建錢包，將 ETH 從加密貨幣交易所傳送到你的 MetaMask 錢包
  • 透過 MetaMask 錢包，在 uniswap v3 官網上，使用 ETH，兌換成 Bananania Coin
  • 購買 Bananania NFT，需透過 MetaMask 錢包，點擊上方 Bananania NFT 連結，使用 ETH 購買

  • 需特別注意，請選用有品牌的加密貨幣交易所，如幣安，MAX，MaiCoin
  • 需特別注意，網址是否正確，如小狐狸錢包 MetaMask : "https://metamask.io/"，加密貨幣交易所 MaiCoin : "https://www.maicoin.com/"
